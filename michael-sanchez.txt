1. What is Semantic HTML?
    Semantic HTML is HTML that provides meaning to the markup. Making HTML semantic means it is easier for humans and machines to interpret the markup.
2. What is HTML used for?
    HTML stands for Hypertext Markup Language, it is used for providing a layout and defining structure for a web document. 
3. What is an attribute and where do we put it?
    An attribute is a value that manipulates elements in HTML. We put them in tags to control elements to operate in a way that meets the developers criteria.
4. What is the h1 tag used for? How many times should I use it on a page?
    An h1 tag is used to designate a header that is the most important of any section on web pages. It is usually toward the top of the web page and should generally only be used once.
5. Name two tags that have required attributes
    <img> tag requires the "src" attribute and the <a> (anchor) tag requires the "href" attribute. 
6. What do we put in the head of our HTML document?
    Metadata goes in the head of the HTML document like the title, navigation, scripts and style sheets.
7. What is an id?
    An id is an HTML tag that is unique identifier to the element in the id tags

10. What is a class?
    A class is an HTML attribute used on any element. It is used by CSS and JS to carry out actions for elements.
11. What elements can I add a class to?
    You can add a class to any elements. 
12. How many times can I use the same class on a page?

14. What is the alt attribute used for?
    The alt attribute is used with img tags. It is used to replace an image with text if the user cannot view the image.

    1. What is the difference between an inline element and a block element?
        inline elements are text based. They don't start new lines, so they can be nested inside other elements because it doesn't disrupt the flow of the content. The "height" and "width" properties have no affect on inline elements. Block elements are structural, which means they start on a new line and take up any space their parent element allows. Control block elements by using the "height" and "width" properties.
